# Sales Analysis Dashboard (Power BI)

## Project Overview

This project presents an interactive Sales Analysis Dashboard built with Power BI to help analyze sales performance across different categories, products, cities, and years.

## Features

- Data Cleaning
- Data Modeling (Star Schema)
- DAX Measures
- KPI Cards
- Interactive Slicers
- Sales Trend Analysis
- Category Analysis
- Top & Bottom Products
- City Sales Analysis

## KPIs

- Total Sales
- Average Sales
- Profit Margin
- Maximum Sales Value

## Tools

- Power BI
- Power Query
- DAX
- Excel
